# 🔗 Link Management Dashboard  

A modern **React + Tailwind** based dashboard UI for managing domains, billing, subscription, invoices, teams, user profiles, and settings.  
This project is modular and includes multiple pages like **Profile**, **Default Settings**, **Geo Templates**, **Campaign Templates**, **Security**, and more.  

---

## 🚀 Features  

- 📊 **Dashboard**  
  - Clicks Analytics  
  - Domain Statistics  
  - Domain Settings  

- ⚙️ **Settings**  
  - Team Management  
  - Billing & Payments  
  - Invoices  
  - Subscription Plans  
  - Profile Management  
  - Plan Information  

- 👤 **Profile Section**  
  - User Information  
  - Default Settings  
  - Geo Templates  
  - Campaign Templates  
  - Security  

- 💳 **Billing & Subscription**  
  - Payment history  
  - Invoice listing  
  - Upgrade/Downgrade subscription  
  - Plan info overview  

- 🛠 **Other Pages**  
  - Help / Documentation  

---

## 🛠️ Tech Stack  

- **Frontend:** React.js, Tailwind CSS, Lucide Icons  
- **State Management:** React Hooks (useState, Context API)  
- **Styling:** Tailwind CSS + Custom components
